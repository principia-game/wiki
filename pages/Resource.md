Placeable objects for various adventure resources, which the robot can pick up in adventure mode.

The different resources that exist are:

- Ruby
- Sapphire
- Emerald
- Topaz
- Diamond
- Copper
- Iron
- Wood
- Aluminium

The amount a given resource object provides can be configured and the resource will shine brighter the more resources it contains.
